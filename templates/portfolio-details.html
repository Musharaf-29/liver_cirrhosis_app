Technical Details

Dataset Information

The model was trained on the Indian Liver Patient Dataset (ILPD) which contains 583 patient records with 10 clinical features.

Feature Importance

The most important features in the model's decision-making process are:

  1. Albumin and Globulin Ratio
  2. Total Proteins
  3. Albumin
  4. Age
  5. Alkaline Phosphotase

Model Architecture

The Random Forest classifier consists of 100 decision trees, with a maximum depth of 10 for each tree. The model uses Gini impurity as the splitting criterion.

Preprocessing

All input features are normalized using a StandardScaler before being fed to the model to ensure consistent scale across features.

Future Improvements

  • Integration with electronic health records for automated data input
  • More comprehensive clinical features
  • Improved model accuracy through ensemble techniques
  • Real-time data visualization
