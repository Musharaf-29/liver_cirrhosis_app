About Liver Cirrhosis Prediction

Project Overview

This web application uses machine learning to predict the likelihood of liver cirrhosis based on clinical test results. The model was trained on a comprehensive dataset of liver function tests and patient outcomes.

About the Application

This tool uses a Random Forest model trained on the Indian Liver Patient Dataset to predict liver cirrhosis stages. Input clinical parameters to get instant, data-driven insights.

How It Works

The prediction system uses a Random Forest classifier that was trained on the following features:

  • Patient age
  • Total Bilirubin levels
  • Direct Bilirubin levels
  • Alkaline Phosphotase levels
  • Alamine Aminotransferase levels
  • Aspartate Aminotransferase levels
  • Total Proteins
  • Albumin levels
  • Albumin and Globulin Ratio

Model Performance

The Random Forest model achieves an accuracy of approximately 68% on the test dataset. While not perfect, it provides valuable insights that can assist healthcare professionals in early detection.
